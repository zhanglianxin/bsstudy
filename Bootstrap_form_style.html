Bootstrap's Default Settings

Individual form controls automatically receive some global styling with Bootstrap.

All texttual <input>, <textarea>, <select> elements with class="form-group" are set to width: 100%; by default.

Standard rules for Bootstrap forms:

  • Add role="form" to the <form> element (helps improve accessibility for people using screen readers)
  • Wrap labels and form controls in a <div> element with class="form-group" (needed for optimum spacing)
  • Add class="form-control" to all textual <input>, <textarea>, and <select> elements

Inline form

Make the viewport larger than 768px wide to see that all of the form elements are inline, left aligned, and the labels are alongside.

Horizontal form

Aligns labels and groups of form controls in a horizontal layout
